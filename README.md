A neon-blue terminal WiFi scanner for Termux. Shows nearby networks with signal strength, band, and security type.

---

## ✨ Features
- 📡 Scans all nearby WiFi networks
- 📶 Signal strength bar + dBm value
- 🔒 Security type (WPA3 / WPA2 / WPA / WEP / OPEN)
- 📻 Band indicator (2.4GHz / 5GHz)
- 🌀 Animated scanning spinner
- 📦 Zero pip dependencies

---

## 📲 Install

```bash
pkg install git python -y
git clone https://github.com/YOUR_USERNAME/wifi-x.git
cd wifi-x
bash install.sh
```

---

## 🚀 Run

```bash
wifiscan
```

---

## 💬 Commands

| Command | Description |
|---------|-------------|
| `/scan` or `s` | Scan networks |
| `/clear` or `c` | Clear screen |
| `/help` or `h` | Show help |
| `/exit` or `q` | Quit |
